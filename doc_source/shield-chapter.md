# AWS Shield<a name="shield-chapter"></a>

AWS provides AWS Shield Standard and AWS Shield Advanced for protection against DDoS attacks\. AWS Shield Standard is automatically included at no extra cost beyond what you already pay for AWS WAF and your other AWS services\. For added protection against DDoS attacks, AWS offers AWS Shield Advanced\. AWS Shield Advanced provides expanded DDoS attack protection for your Amazon Elastic Compute Cloud instances, Elastic Load Balancing load balancers, Amazon CloudFront distributions, Amazon RouteÂ 53 hosted zones, and your AWS Global Accelerator accelerators\.
